# Corebridge Financial (corebridge-financial)

Corebridge Financial (formerly AIG Life and Retirement) is one of the largest and most established retirement and insurance solutions providers in the United States. Spun off from AIG in September 2022 in the largest IPO of that year, Corebridge provides retirement solutions, life insurance, fixed and variable annuities, and institutional products. The company is a leading retirement plan provider for K-12 schools, healthcare, government, higher education, and other not-for-profit institutions.

## APIs

### Corebridge Financial Website

Corebridge Financial does not currently publish a public developer API. The company provides authenticated portals for participants, plan sponsors (SponsorFIT), and consultants (ConsultantDIRECT) but no publicly documented REST APIs.

Note: Corebridge Financial is a separate entity from CoreBridge V3 software (corebridge.net) which has its own API portal.
